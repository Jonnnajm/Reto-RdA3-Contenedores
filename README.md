# Reto RdA3 Contenedores

## Anatomía de un Contenedor: Del Kernel Linux a WordPress en Docker

Este proyecto implementa un sistema web WordPress utilizando Docker y Docker Compose, con el objetivo de analizar los mecanismos del kernel Linux que hacen posible la tecnología de contenedores.

## Requisitos
- Docker
- Docker Compose (plugin)

## Despliegue

docker compose up -d
